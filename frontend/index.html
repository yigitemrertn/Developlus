Qwen 2.5 LLM

Alibaba'nın güçlü Qwen 2.5 modeli ile doğal dil anlama ve üretimde üstün performans. 72B parametre, Türkçe dahil çok dil desteği.

LiteLLM Proxy

RAG & Vektör Arama

Belgelerinizi yükleyin, pgvector HNSW indeksiyle anlık semantic arama yapın. PDF, TXT, Markdown formatları desteklenir.

pgvector + HNSW

Real-time Streaming

SSE (Server-Sent Events) ile token-by-token gerçek zamanlı yanıt akışı. Gecikme hissi sıfır, kullanıcı deneyimi maksimum.

SSE / FastAPI

Güvenli Auth

JWT + Refresh Token ile güvenli kimlik doğrulama. Bcrypt şifrelemesi, rate limiting ve CORS koruması dahil.

JWT / Bcrypt

Akıllı Cache

Redis semantic cache ile benzer sorular tekrar LLM'e gönderilmez. API maliyetlerini %30-60 azaltın.

Redis / Semantic Cache

Mikroservis Mimarisi

Docker Compose ile tam izole servisler: API, Worker, LiteLLM Proxy, PostgreSQL, Redis. Üretim için Kubernetes hazır.

Docker / K8s
